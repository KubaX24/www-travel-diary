# Cestovatelský deník

Webová aplikace pro cestovatele umožňující zaznamenávat místa s počtem nachozených kilometrů. Místa se mohou zaznamenat do několik deníků, které umožňují si nastavit cíl nachozených kilomentrů.

## Zdroje dat a knihoven
- Mapové dlaždice: [Mapy.com](https://developer.mapy.com/r)
- Knihovna pro práci s dlaždicemi: [MapLibre](https://maplibre.org/)
- Ikonky: [FontAwesome](https://fontawesome.com/)
